# searchForm react.js

This form allows the users to enter search terms and see 10 most relevant results.
[API](https://api-search.win.gg/search?q=start&index=tournament)

# Usage

```bash
npm install
```

Or, if yarn is your thing:

```bash
yarn install
yarn dev
open http://localhost:3030
```

# Run flow

```bash
yarn flow
```
